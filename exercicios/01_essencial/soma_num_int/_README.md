## Objetivo

Este é um exercício introdutório para praticar a leitura de valores e operações matemáticas básicas. Vamos trabalhar com a soma de dois números inteiros.

## Especificação

### Some dois números inteiros

- Abra o arquivo `main.py`. Dentro dele, você encontrará a função `resposta`.
- Essa função deverá receber dois números e retornar a soma desses dois valores.
- O valor retornado deverá ser um inteiro.

Atenção: utilize return, não print.
