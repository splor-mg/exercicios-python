## Objetivo

Este exercício reforça o uso de funções e operações matemáticas básicas.

## Especificação

### Some dois números

Abra o arquivo `main.py`. Dentro dele, localize a função `resposta`.

A função deve receber dois números inteiros.

Sua tarefa é retornar a soma desses dois valores.

Regras:

- O resultado deve ser um número inteiro

Exemplos:

- `resposta(2, 3)` → `5`
- `resposta(-1, 1)` → `0`

**Atenção:** utilize `return`, não `print`.
