## Objetivo

Este é um exercício introdutório para praticar a leitura de valores e operações matemáticas básicas. 

## Especificação

### Transforme uma data para segundos

- Abra o arquivo `main.py`. Dentro dele, localize a função resposta. Essa função deverá receber:

- um `int` correspondente ao dia
- um `int` correspondente à hora
- um `int` correspondente aos minutos
- um `int` correspondente aos segundos

Sua tarefa é retornar o total de segundos correspondente ao valor data.

Atenção: utilize return, não print.
