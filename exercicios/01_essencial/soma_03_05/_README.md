## Objetivo

Neste exercício, você irá praticar o uso de laços de repetição e condições para resolver um problema matemático.

## Especificação

### Some os múltiplos de 3 ou 5

Abra o arquivo `main.py`. Dentro dele, localize a função `resposta`.

A função deve receber um inteiro positivo `n`.

Sua tarefa é retornar a soma de todos os números menores que `n` que sejam múltiplos de 3 ou 5.

Regras:

- Considere apenas números menores que `n`
- Inclua múltiplos de 3, de 5 ou de ambos

Exemplos:

- `resposta(5)` → `3`
- `resposta(10)` → `23`
- `resposta(20)` → `78`

**Atenção:** utilize `return`, não `print`.
